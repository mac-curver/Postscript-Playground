Benutzung (Beispiele zum Editieren)

Nach dem Starten der Anwendung wird im linken Editorbereich ein kleines Postscript Programm angezeigt.

Im linken Teil des Fensters können Postscript Kommandos eingegeben werden. Die rechte Seite des Fensters kann nur vergrößert, verkleinert oder verschoben werden. Ein Rechtsklick in diesem Bereich zeigte einige Optionen zum Verändern des gerenderten Bildes und als Option, die PDF Datei auf einem Laufwerk zu speichern.

Some first steps

By pressing the blue play button at the bottom of the window a "Write to ps" file save dialog will be displayed. Please define a file name and select the storage place in the file dialog. An explicit file dialog is necessary as the application should run in a Sandbox and can't access the file system directly**.

After having pressed the Save button the Postscript file will be rendered and displayed on the right hand side in the PDF preview.

Some edits

Replace the "50" font scale on line 16 by typing 100 over it. After pressing the play button the text on the right hand side will be increased.

Automatic mode

Similar to the Xcode playground you can switch to automatic mode by selecting Automatically run holding the small triangle besides the Play button.

Select the "4" in front of the setLineWidth expression and replace it by 9. The number will be displayed in black, changes to the number color and the width of the underline increases.

Errors

Replace the 9 by letter "B". This provokes an error and the Postscript file can't be rendered. In this case a yellow exclamation mark is blinking at the right upper corner of the PDF section.

Clicking onto the exclamation mark will display the error message: "Postscript file couldn't be converted". In this case the letter B is the offending command indicated in the error message "OffendingCommand: B; "

Correct the error by replacing the letter B with "4" again and the file will be rendered again.

)* The file is also available as example file.

)** Currently sandbox is not supported as it would not allow using an external tool.
