Knowledge

Please find here some expert modifications.

  • Reading / deleting preferences:

    Enter the following commands into Terminal:
    Reading

    defaults read de.legoesprit.postscript-playground

    Deleting

    defaults delete de.legoesprit.postscript-playground

  • Checking console message with Console.app (Admin account required):

    Open the Console.app application and enter

    de.legoesprit.postscript-playground

    into the search field. If questioned enter an admin user and his password. Change the context popup menu in the search button to "subsystem".
    The messages are visible only for short periods of time!

  • Checking console message with Terminal.app (Sudoer required):

    Run

    sudo log stream --predicate 'subsystem == "de.LegoEsprit.Postscript-Playground"' | awk 'NR>2 {print substr($0,12,15) " T:" substr($0,index($0,"00 ")+3, 8) substr($0, index($0,"]")+1) }'

    from within Terminal. The command line streams the current log of the application and pipes it towards awk. The awk command skips the first 2 lines and copies only the required columns into the output window.
