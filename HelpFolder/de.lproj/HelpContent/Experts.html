Wissen

Für Experten.

  • Lesen und löschen der Präferenzen:

    Bitte tippen Sie folgende Kommandos im Terminal:
    zum Lesen

    defaults read de.legoesprit.postscript-playground

    zum Löschen

    defaults delete de.legoesprit.postscript-playground

  • Konsolennachrichten mit der Konsole.app überprüfen(Admin Benutzerrechte sind notwendig):

    Öffnen Sie die Konsole.app und tippen Sie

    de.legoesprit.postscript-playground

    ins Suchfeld. Wenn gefragt geben Sie den admin Benutzer und sein Passwort ein. Im Suchfeld ändern Sie den Eintrag im Popupmenü auf "subsystem".
    Die angezeigten Mitteilungen bleiben nur eine gewisse Zeit erhalten!

  • Konsolennachrichten mit der Terminal.app überprüfen (Der Benutzer muss als Sudoer registriert sein):

    Geben Sie

    sudo log stream --predicate 'subsystem == "de.LegoEsprit.Postscript-Playground"' | awk 'NR>2 {print substr($0,12,15) " T:" substr($0,index($0,"00 ")+3, 8) substr($0, index($0,"]")+1) }'

    im Terminal ein. Diese Kommandozeile streamt kontinuierlich die log Datei der Applikation und sendet sie an awk. Das awk Kommando überspringt die ersten 2 Zeilen und kopiert nur die relevanten Spalten in das Ausgabefenster.
